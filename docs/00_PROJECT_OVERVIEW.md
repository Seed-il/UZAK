# UZAK Project Overview

## 프로젝트 이름
UZAK(한글 서비스 이름 : 우작)

## 프로젝트 목적
이 프로젝트는 하네스형 바이브 코딩 방식으로 개발하는 실험/실전 프로젝트다.

## 만들고 싶은 서비스
슬로건(Slogan) : "우리는 작가다."
서비스 설명 : 모든 사람은 독자이며 작가이다. 이 서비스에서 누구나 웹소설을 감상할 수 있고, 연재할 수 있다.

## 핵심 목표
- AI 역할 분리
- 문서 기반 개발
- 작업 티켓 기반 구현
- 테스트와 리뷰 루프 구축

## 사용 기술
※ 아래 기능 중 피드, 결제, 구독, 관리자 기능은 MVP 이후 확장 기능으로 분리한다.
- Frontend: 홈/카테고리/피드/세팅 4가지 하단 메뉴 구성
홈 : 공지사항/이벤트/추천 작품
카테고리 : 카테고리 별 최신순/인기순 작품 리스트, 감상 중인 작품 등 표시
피드 : 팔로우 중인 회원의 실시간 피드 확인 기능 및 피드 작성 기능
세팅 : 계정 정보 세팅
- Backend: 회원가입 / 로그인 / 탈퇴 / 작품 리딩 / 작품 업로딩 / 구매 및 유료 구독 / 팔로잉 / 피드 작성, 수정, 삭제 / 관리자 권한- 모니터링 및 회원 제재 / 개인 정보 암호화
- Database: 회원 관리 / 작품 관리 / 피드 관리 / 카테고리 관리 / 페이지 관리 / 구매내역 관리 / 개인정보 암호화 저장

## 초기 버전 범위 (MVP)
- 회원가입 / 로그인
- 작품 목록
- 작품 상세
- 글 작성

## 향후 확장
- 결제 / 구독
- 피드
- 팔로우
- 관리자 기능

## 개발 방식
하네스형 바이브 코딩
